# HR Assistant MCP

An intelligent HR Assistant project built with Python and enhanced using the Model Context Protocol (MCP).

## Overview

This project originally started as a basic HR assistant without MCP integration.  
The first version focused on handling simple HR-related tasks such as:

- Employee information management
- Basic HR question answering
- Internal assistance workflows
- Simple AI-powered responses

Later, the project was upgraded by integrating **MCP (Model Context Protocol)** to make the assistant more modular, scalable, and capable of interacting with external tools and services dynamically.

---

# Why MCP?

The goal of adding MCP was to transform the assistant from a simple standalone chatbot into a smarter AI system capable of:

- Connecting with external tools
- Handling structured workflows
- Improving context management
- Supporting scalable integrations
- Making future feature expansion easier

MCP allows the assistant to communicate with tools in a clean and organized architecture instead of relying only on hardcoded logic.
